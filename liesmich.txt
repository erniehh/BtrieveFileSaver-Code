Btrieve File Saver 
------------------

Btrieve File Saver ist ein daten Wiederherstellungstool verfügbar für 
windows 32/64 Bit und als source inklusive make file zum erstellen
einer linux version

Hauptfeature von Btrieve File Saver:
   - data wiederherstellung von btrieve Dateien der Version 3/4/5/6/7/8/9.0 (9.5 geplahnt)
   - liest und dump's Daten selbst wenn Btrieve selbst die Daten nicht mehr lesen kann
   - unterstützt variable Datensätze
   - unterstützt alle möglichen page size Grössen
   - Batch integration möglich
   - Silent Mode (arbeitet in Batchmode ohne notwendige interaktion)
   - CRC support zur Datenverifizierung
   - Speichert Daten in BUTIL -save Format oder HEX (mit oder ohne Längenangaben oder CRC Information)
